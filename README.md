Strike is a single-player baseball batting game in virtual reality. In this game, baseball is
simulated in an arcade fashion where it is easy-to-pick-up with fun and intuitive gameplay. All
the rules and skills of baseball are included in a simplified manner, so the player can focus on
having fun. This project hopes to provide a seamless introduction to baseball without the
boundaries of gathering people for organized play
